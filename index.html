About This Project

This website introduces the fundamentals of Information Technology (IT), focusing on the relationship between software and hardware. Created as a college research project, it aims to simplify complex concepts for beginners.

Software

Software is the digital component of a computer system — the programs and operating systems that allow hardware to perform tasks.

  • Operating Systems: Windows, macOS, Linux
  • Applications: Browsers, Office tools, Media players
  • Development Tools: VS Code, Python, C++

Hardware

Hardware includes the physical components of a computer system — the parts you can touch and see. These components work together to process data and run software.

  • CPU: The brain of the computer
  • GPU: Handles graphics and visual processing
  • RAM: Short‑term memory for active tasks
  • Storage: SSDs and HDDs for long‑term data
  • Networking: Routers, switches, cables
